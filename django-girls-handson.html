django girls のハンズオンで、Python + Pelican を使ってサイトを作成してみましたよ！

Sat 19 December 2015


ども、なのまるです！

Pythonを使った、静的サイトジェネレータ「Pelican」を使ってサイトを作るワークショップに参加しました！

Pelican documentation

今回は、自分のVPSでUbuntuとPythonで設定しました！

sudo pip install pelican
mkdir pelican-demo
pelican-quickstart
vim content/test.md
make html
python -m SimpleHTTPServer

これで、 http://localhost:8000/ で、アクセスできました。

あとは、content の下に abcd.md などを作って「make html」するだけでページが公開できました！
